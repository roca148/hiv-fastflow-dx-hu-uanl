# HIV FastFlow Dx | HU-UANL

Algoritmo interactivo educativo para diagnóstico de VIH basado en guías internacionales recientes (OMS, CDC/APHL, CENSIDA) y contexto clínico mexicano. Desarrollado para el Servicio de Inmunología del Hospital Universitario Dr. José Eleuterio González, UANL.

## Características
- Algoritmo diagnóstico basado en NOM-010-SSA-2023 y guías OMS/CDC 2024-2025
- Rutas pediátricas, profilaxis post-exposición (PEP) y tamizaje de infección aguda
- Integración de algoritmo OMS (3 pruebas consecutivas) y enfoque CDC/APHL (4ª gen Ag/Ac)
- Casos clínicos simulados para entrenamiento médico
- Funciona completamente en el navegador, sin conexión a servidores

## Tecnología
- HTML5 / CSS3 / JavaScript vanilla
- Sin dependencias externas
- Compatible con navegadores modernos

## Referencias normativas
- NOM-010-SSA2-2023 (CENSIDA, México)
- WHO HIV Testing Guidelines 2024
- CDC/APHL HIV Diagnostic Algorithm 2025
- NIH Guidelines for HIV Diagnosis
